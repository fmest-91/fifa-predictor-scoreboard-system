🏆 Connect Your Google Sheets API

To connect your live prediction league, enter the Google Apps Script Web App URL deployed from your Google Sheet.

Ensure you deployed the Apps Script as a Web App, executed as 'Me', with access set to 'Anyone'.

Add the public link to your Google Form so players can access it directly from the header button.

How to setup your Google Sheet backend:

  1. Create a new spreadsheet on Google Sheets.
  2. Go to Extensions > Apps Script.
  3. Copy and paste the code from google-apps-script/Code.gs.
  4. Save and run the initialSetup function to build sheets and seed matches.
  5. Run generatePredictorForm to generate the Google Form.
  6. Click Deploy > New deployment, choose Web App, set Executed as: Me, Who has access: Anyone, and copy the deployment URL!
